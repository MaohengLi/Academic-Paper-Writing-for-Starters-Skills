## English

### Overview

A comprehensive guide for academic paper writing and formatting in English, based on common issues identified across **100+ papers**. This skill covers language level problems, style guidelines, organization, and tools to help researchers write publication-ready papers.

### Features

- **Language Issues**: Active voice, pronoun usage, tense rules, common mistakes
- **Style Guidelines**: Formal writing rules, word choice, consistency requirements
- **Organization**: Section guidelines, visual elements, mathematical notation
- **Tools**: LaTeX vs Word, drawing tools, bibliography management

- 40 language rules

✅ **Based on real data** from Academic Paper Writing for Starters by 徐悦甡 (Xidian University)

✅ **Practical examples** for common pitfalls and how to avoid them

#### Example Prompts

```
"Check my paper for academic writing issues"
"How should I write the abstract?"
"Fix the passive voice in my introduction"
"Is 'this paper' repeated too often?"
"Help me format my bibliography"
"Improve the clarity of this paragraph"
```

### Key Principles

1. **Use Active Voice**: In abstract and contributions, use "We propose..." instead of passive constructions
2. **Be Precise**: Replace ambiguous pronouns (This/It) with specific nouns (This method/our approach)
3. **No Chinese Punctuation**: Use English punctuation only (., not 。、）
4. **Consistency**: Use the same terminology throughout the entire paper
5. **Clarity Over Complexity**: Write for the reader, not to impress

### Common Fixes

| Problem | Fix |
|---------|-----|
| "can not" | "cannot" |
| "find out" | "find" |
| "very important" | "vital" / "crucial" |
| "This suggests..." | "This result suggests..." |
| "It provides..." | "The proposed method provides..." |
| Chinese punctuation | English punctuation |
| "and so on" | List explicitly or use formal alternatives |

Before submitting your paper:
- [ ] Spell check enabled and run
- [ ] All Chinese punctuation removed
- [ ] "we" usage appropriate (active in abstract/contributions)
- [ ] Tenses correct (past in Related Work, present elsewhere)
- [ ] No contractions (don't, can't, it's)
- [ ] All abbreviations defined on first use
- [ ] Terminology consistent throughout
- [ ] No ambiguous pronouns (This/It without clear reference)

### 使用方法

#### 触发技能

在询问以下主题时使用此技能：
- 学术写作 (academic writing)
- 论文格式 (paper formatting)
- LaTeX
- 研究论文 (research paper)
- 学位论文 (thesis writing)
- 手稿准备 (manuscript preparation)
- 摘要、引言、相关工作、结论部分

#### 示例提示

```
"检查我的论文是否存在学术写作问题"
"摘要应该如何写？"
"修复引言中的被动语态"
"'this paper'是否重复太多次？"
"帮助我格式化参考文献"
"提高这段话的清晰度"
```

### 核心原则

1. **使用主动语态**：在摘要和贡献部分使用"We propose..."而非被动结构
2. **精确表达**：用具体名词替换模糊代词（This/It → This method/our approach）
3. **无中文标点**：仅使用英文标点（用。而非。，）
4. **保持一致**：整篇论文使用相同的术语
5. **清晰优先**：为读者而写，而非为了炫技

### 常见修正

| 问题 | 修正 |
|------|------|
| "can not" | "cannot" |
| "find out" | "find" |
| "very important" | "vital" / "crucial" |
| "This suggests..." | "This result suggests..." |
| "It provides..." | "The proposed method provides..." |
| 中文标点 | 英文标点 |
| "and so on" | 明确列出或使用正式替代表达 |
